# Geometry Dash Web Engine (TBM V0.2)

🔥 A Geometry Dash-like engine running in browser.

## Features
- Cube movement
- Gravity physics
- Jump system
- Camera follow
- Spike collision
- Death & respawn

## Controls
- SPACE: Jump
- Auto run → right
